# Saklı Eşya Avı

Tarayıcıda çalışan, 20 seviyeli bir saklı eşya bulma oyunu.

## Özellikler
- Toplam **20 kademe** bulunur.
- Seviye arttıkça:
  - Izgara büyür,
  - Bulunacak eşya sayısı artar,
  - Verilen süre azalır.
- Oda temaları gündelik yaşam mekanlarından seçilir:
  - Ev
  - Okul
  - İşyeri
  - Laboratuvar
  - Spor salonu
  - Alışveriş merkezi
  - Market

## Çalıştırma
Statik dosya olduğu için doğrudan `index.html` açılabilir.

Yerel sunucu ile çalıştırmak için:

```bash
python3 -m http.server 4173
```

Sonra tarayıcıda `http://localhost:4173` adresine gidin.
